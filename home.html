Welcome to Flow...

An attempt at hosting a FluidApp for writing FluidApps within FluidDB. Lots of stuff doesn't work and I'm nowhere near finished. Your mileage may vary but stuff is being added all the time. Comments and feedback most welcome.

FluidDB is a product of FluidInfo. Flow is in no way affiliated with FluidInfo. Rather, it is the result of eager alpha-testing... ;-)

In order to use this software you should request a username and password for alpha testing at the following address:

http://fluidinfo.com/accounts/new

Flow makes use of the following third party libraries:

  • The Sammy javascript framework
  • The jsFluidDB library.
  • The Codemirror javascript in-browser code editor.
  • jQuery
